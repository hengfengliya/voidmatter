# VM · 虚空有物官方网站

> 于未知处，见未来值 - VM（上海虚空有物科技有限公司）官方网站

## 🌌 项目概述

这是VM（上海虚空有物科技有限公司）的多业务平台，包含：

1. **主官网** - 公司品牌展示网站（根目录）
2. **质点智库** - 思想领导力内容平台（think-tank.html）
3. **行业研究报告系统** - 专业投研内容管理（行业投研报告/目录）

### 核心理念
- **虚空 (Void)**: 潜能的策源地，市场的非有效性和前沿科技的萌芽阶段
- **有物 (Substance)**: 价值的最终形态，具备长期确定性的价值载体
- **认知套利**: 系统性地将"认知价差"转化为客户的"认知Alpha"

## 🎨 设计特色

### 视觉哲学 - Dark Matter Design
- **主色调**: 纯黑背景 (#0f0f0f)，营造深邃的视觉体验
- **点缀色**: 金色 (#FFD700) 和深金色 (#B8860B)，体现财富和价值的象征
- **字体**: 中文使用思源黑体，英文使用Inter和Playfair Display的组合
- **布局**: 高对比度 + 大留白 + 巨构字型，形成视觉张力

### 交互体验
- **金融呼吸感**: 所有动画都具有缓慢、优雅的节奏
- **视差滚动**: 分段式的视差效果，增强沉浸感
- **微交互**: 鼠标悬停、点击等细节动效

## 🛠️ 技术架构

### 前端技术栈
- **HTML5**: 语义化结构，良好的SEO支持
- **TailwindCSS 4.1+**: 原子化CSS框架，快速构建响应式布局
- **Anime.js**: 轻量级动画库，实现流畅的页面动效
- **Apache ECharts 5**: 专业的数据可视化图表库
- **Font Awesome**: 图标库
- **Google Analytics 4**: 网站流量分析和用户行为追踪

### 核心系统模块

#### 1. 主网站系统 (index.html)
- 英雄区域动画展示
- 认知套利图表可视化
- 投资哲学流程展示
- 关注领域矩阵
- 联系交互功能

#### 2. 智库系统 (think-tank.html)
- **会员认证**: 基于sessionStorage的前端认证
- **分类筛选**: 中信行业分类体系，支持四级分类
- **报告管理**: 动态报告卡片系统
- **权限控制**: 会员专享内容访问控制

#### 3. 报告系统 (行业投研报告/)
- **标准化命名**: `{行业编码}{行业名称}行业研究报告_YYYYMMDD.{md|html}`
- **双格式输出**: Markdown源文件 + HTML展示文件
- **深邃黑金UI**: 权力、财富、稀缺、掌控设计理念
- **垂直进度条**: 支持悬停显示标题和点击跳转

## 📊 HTML报告生成规范

### 🚨 **严格要求 - 必须遵守**

#### 格式规范
- **禁止编号格式**: 严禁使用"第一章"、"第二章"、"1.1"、"1.2"等任何编号格式
- **内容居中对齐**: 确保所有标题和内容元素正确居中显示，避免额外空格导致的偏移
- **完整内容还原**: 不删改、不省略报告内容，仅为美观优化层级和样式
- **禁用Markdown语法**: 不使用markdown语法符号（如**、##等），直接使用HTML标签和CSS类

#### UI设计要求
- **色彩系统**: 严格使用深邃黑金权力色系（--obsidian-black, --sovereign-gold等）
- **垂直进度条**: 必须包含右侧垂直进度条导航，支持悬停显示标题和点击跳转
- **响应式布局**: 支持不同屏幕尺寸的自适应显示
- **交互元素**: 支持悬停效果和平滑滚动功能

#### 技术实现要求
- **样式参考**: 严格1:1参考 `行业投研报告/doc/UI报告样式参考.md`
- **CSS框架**: 使用TailwindCSS + Material Icons
- **文件命名**: 与MD文件名保持一致
- **发布时间**: HTML内的发布时间与MD文件名中的发布时间一致

#### 内容规范
- **标题格式**: 大标题为MD里的标题+年份季度（如2025年Q3）
- **引用文献**: 必须使用固定4列布局，完整呈现全部引用内容
- **章节标识**: 为每个章节添加data-title属性，用于进度条功能
- **链接完整性**: 为所有引用文献提供可点击链接
